# xdump

## Purpose

Dump all the variables in the **ord**th context node and their values to** fileName.**

## Syntax

```
xdump(fileName, ord)
```

```
fileName - type: str

ord - type:
```

## Returns

## Remarks

The output file (**fileName**) must be set up in advance using **openfile**, **fileout** or an output statement.
