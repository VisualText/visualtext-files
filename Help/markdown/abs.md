# abs

## Purpose

Compute the absolute value of an integer.

## Syntax

```
returnedInt = abs(numInt)
```

```
returnedInt - type: int

numInt - type: int
```

## Returns

The absolute (non-negative) value of the given number.

## Remarks

If the number is negative it is negated; otherwise it is returned unchanged.

## Example

```
@CODE

"output.txt" << abs(-7) << "\n";

@@CODE
```

```
Outputs:
```

```
7
```
