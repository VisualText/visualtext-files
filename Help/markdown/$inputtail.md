# $inputtail

## Purpose

Get the tail (extension type) of the input file.

## Syntax

```
returnedString = variableType("$inputtail")
```

```
returnedString - type: str

variableType - type: G
```

## Returns

## Remarks

## Example

$inputtail returns "txt" if the input file is "rez.txt".
