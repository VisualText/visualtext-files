# option

## Purpose

Match a minimum of 0 and a maximum of 1 node. Indicates element is optional.

## Remarks

Short for min=0 max=1.  **opt**, **optional** and **o** are alternate forms for option.

## Example

```
_vgroup <- _modal [option]
 _have [option] _be [option]
 _verb @@
```
