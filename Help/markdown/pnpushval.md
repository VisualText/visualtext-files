# pnpushval

## Purpose

Push a value onto the front of pnode's variable.

## Syntax

```
None = pnpushval(pnode, var_str, val);
```

```
pnode - type: pnode

var_str - type: str

val - type: str, int, flt, con, phrase, attr, val, or pnode
```

## Returns

None

## Remarks

New in 2.7.3.2.
