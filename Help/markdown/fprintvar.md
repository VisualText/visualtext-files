# fprintvar

## Purpose

Print the values of the **global** variable **varName** to **fileName**.

## Syntax

```
fprintvar(fileName, varName)
```

```
fileName - type: str

varName - type: str
```

## Returns

## Remarks

Old-style.  Use NLP++ syntax such as

"output.txt" << G("varname") << "\n";
