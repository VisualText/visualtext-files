**Running the Analyzer on Text**

![](../../../../helps/Tutorials/GO.gif) First, select the text you want to process, in the Text Tab:

![](../../../../helps/HowToBuild/Tour15/Run/Image156.gif)

![](../../../../helps/Tutorials/GO.gif) To view the text, either double-click on the text icon (above) in the Text Tab, or go to the Debug Toolbar and click on the "paper" button:

![](../../../../helps/HowToBuild/Tour15/Run/Image160.gif)

The text window pops up in the workspace:

![](../../../../helps/HowToBuild/Tour15/Run/Image161.gif)

**The Run Button**

![](../../../../helps/Tutorials/GO.gif) To run the analyzer on the selected input text, choose the running-man button in the Workspace Toolbar:

![](../../../../helps/HowToBuild/Tour15/Run/Image157.gif)

The cursor becomes an hour glass and the stop button is activated:

![](../../../../helps/HowToBuild/Tour15/Run/Image162.gif)

The Log Window will display stats about the run, including any syntax errors it discovered:

![](../../../../helps/HowToBuild/Tour15/Run/Image158.gif)

You have successfully run the corporate analyzer on our "Sold.txt".
