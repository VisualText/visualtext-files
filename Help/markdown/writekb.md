# writekb

## Purpose

Dumps the entire KB currently in memory to the kb/user directory which includes a word.kb, attrs.kb, and main.kb file.

## Syntax

```
returnedBoolean
 = writekb()
```

## Returns

Returns **1** is true and **0** if false.
