# $islastfile

## Purpose

Checks to see if the current file is the last file being analzyed in a directory.

## Syntax

```
returnedBoolean = variableType("$islastfile")
```

```
returnedBoolean - type: int

variableType - type: G
```

## Returns

Returns 1 if the current file is the last file, 0 if not.
