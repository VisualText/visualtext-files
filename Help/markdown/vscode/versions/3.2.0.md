# What's New in 3.2.0

This release brings a built-in help system, golden-file regression testing, and a
simpler Python pass.

[← Back to Help home](../home.md)

## 📚 In-extension help

- A new **Help** view in the NLP++ sidebar and a 📖 book button on the view toolbars
  open this help hub.
- **Version notes** (like this page) appear automatically the first time you run a new
  significant release, and are listed on the [Help home](../home.md) page.

## 🧪 Regression testing

- **Run Regression Test** and **Bless Regression Goldens** on an analyzer (Analyzers
  view) or on a single file/folder (Text view 🧪 icon / right-click).
- Compares your analyzer's `output.json` against committed goldens under
  `test/expected/`, semantically (ignores `id`, order-insensitive).
- Results stream into the **Logging** view. See **[Regression Testing](../testing.md)**
  for setup (your analyzer must emit `output.json` via `KBFuncs`).

## 🐍 Python passes

- A single **Python** pass type that runs wherever you place it in the sequence — put it
  **before the tokenizer** to run on raw text, or after it to run post-tokenization.
- Insert from the Analyzer (sequence) view; the tokenizer's menu has
  *Insert Python Pass Before Tokenizer*.

## 🛠 Compiling

- **Compile Analyzer Only** produces `analyzer.dll`; the run-mode toggle now cycles
  Interpreted → Compiled KB → Compiled Analyzer → Compiled. See
  **[Compiling Analyzers](../compiling.md)**.
