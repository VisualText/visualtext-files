# 📢 Announcement — NLP++ Grows Up: A Full IDE in VS Code

**Version 3 turns VS Code into a genuine home for building analyzers with NLP++** —
from your first rule to a compiled analyzer shipped to a customer.

Version 3 began by making analyzers **compiled to native C++**, **built in the cloud** for
every platform, and **available on npm**. It has since gained the full code-intelligence
experience you'd expect from a mainstream language:

- 🧭 **Navigate** — outline, go-to-definition (across passes and into `.kbb` concepts),
  find references, rename, and workspace symbol search.
- 🚦 **Diagnostics** — a structural linter plus inline squiggles mapped straight from the
  engine's `err.log` to the offending pass or dictionary line.
- ⌨️ **Write faster** — context-aware IntelliSense and function signature help.
- 👀 **Read easier** — code folding, semantic highlighting, and one-click quick fixes for
  misspelled function calls.

All of it sits on top of the Version 3 foundation: a lossless NLP++ formatter, an
in-extension help system, a native regression-test runner, and cloud compile by default
(no local C++ toolchain required).
