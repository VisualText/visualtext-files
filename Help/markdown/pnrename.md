# pnrename

## Purpose

Rename a given parse tree node *aPnode*. This function returns the interned name.

## Syntax

```
returnedString = pnrename(aPnode, newNameString)
```

```
returnedString - type: str

aPnode - type: pnode

newNameString - type: str
```
