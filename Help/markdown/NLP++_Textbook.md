## The First Textbook on the NLP++ Programming Language

The first textbook on NLP++ is now available world-wide through [BPB Online](https://book.visualtext.org).

NLP++ can replace LLMs when used in agentic flows. The code must be written by a human like any
other programming language, and this book will facilitate that process. NLP++ is not a statistical
system that needs training — it relies on the ingenuity of the programmer to create a program that
can parse text and extract information in a deterministic way.
