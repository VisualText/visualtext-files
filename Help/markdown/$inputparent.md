# $inputparent

## Purpose

Get the name of the parent directory of the input file.

## Syntax

```
returnedString = variableType("$inputparent")
```

```
returnedString - type: str

variableType - type: G
```

## Returns

## Remarks

## Example

$inputparent returns "Dev1" assuming input file is "D\apps\Resume\input\Dev1\text.txt".
