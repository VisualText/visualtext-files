# strcontainsnocase

## Purpose

Check if string 1 is contained in string 2, ignoring letter case.

## Syntax

```
returnedBoolean = strcontainsnocase(wordString1, wordString2)
```

```
returnedBoolean - type: bool

wordString1 - type: str

wordString2 - type: str
```
