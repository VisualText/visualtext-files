# layers

## Purpose

Layer additional attributes for the element in the parse tree, as "mini-reductions".

## Remarks

Use the names in the list to name nodes. Each node that matched current rule element will be layered.  **layer**, **attrs**, and **attr** are alternate forms for layers.

**layers** can also be used as a suggested element modifier.
