# nodeowner

## Purpose

Fetch the concept that owns the given node's phrase.

## Syntax

```
returnedConcept = nodeconcept(node)
```

```
returnedConcept - type: con

node - type: con
```

This usurps the functionality formerly given to the *nodeconcept* function.
