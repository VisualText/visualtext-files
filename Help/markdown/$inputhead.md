# $inputhead

## Purpose

Get the head (name) of the input file.

## Syntax

```
returnedString = variableType("$inputhead")
```

```
returnedString - type: str

variableType - type: G
```

## Returns

## Remarks

## Example

$inputhead returns "rez" if the input file is "rez.txt".
