# fprintgvar

## Purpose

**OBSOLETE.**  To **fileName**, print the value of the global variable **var**.

## Syntax

```
fprintgvar(fileName, var)
```

```
fileName - type: str

var - type:
```

## Returns

## Remarks

The output file (**fileName**) must be set up in advance using **openfile**, **fileout** or an output statement.
