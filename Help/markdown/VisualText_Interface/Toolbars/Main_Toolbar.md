# Main Toolbar

![](../../../helps/VisualText_Interface/Toolbars/mainToolbar.gif)

The Main Toolbar provides access to commonly used functions such as opening, closing and saving files. It also contains the controls for toggling the Main Windows.

| **Button** | **Name** | **Description** |
| --- | --- | --- |
| ![New](../../../helps/Toolbar_buttons/new.gif) | New | Creates a new file and displays it in the Workspace window. |
| ![Open](../../../helps/Toolbar_buttons/open_button.gif) | Open | Opens a file and displays it in the Workspace. |
| ![Save](../../../helps/Toolbar_buttons/save_button.gif) | **Save** | Saves currently selected file in the Workspace. |
| ![Save All](../../../helps/Toolbar_buttons/save_all_button.gif) | Save All | Saves all unsaved files in the Workspace. |
| ![Cut](../../../helps/Toolbar_buttons/cut_button.gif) | Cut | Deletes selected object in a Workspace file. |
| ![Copy](../../../helps/Toolbar_buttons/copy_button.gif) | Copy | Copies selected object to the clipboard. |
| ![Paste](../../../helps/Toolbar_buttons/paste_button.gif) | Paste | Pastes object from the clipboard at insertion point in currently selected file. |
| ![Print](../../../helps/Toolbar_buttons/print_button.gif) | Print | Prints currently selected file in the Workspace. |
| ![Find in Files](../../../helps/Toolbar_buttons/find_in_files_button.gif) | Find in Files | Launches Find in Files dialog box to search for items across multiple files. File type can be specified. Search results displayed in the Find Window. |
| ![Find Previous](../../../helps/Toolbar_buttons/find_previous_button.gif) | Find Previous | Searches for the last occurrence of an item in the find dialog. |
| ![Find Next](../../../helps/Toolbar_buttons/find_next_button.gif) | Find Next | Searches for the next occurrence of an item in the find dialog. |
| ![Toggle Tab Window](../../../helps/Toolbar_buttons/toggle_tab_win_button.gif) | Tab Window | Toggles the visibility of the Tab Window. By default, the Tab Window is located in the middle left portion of the interface. |
| ![Toggle Log Window](../../../helps/Toolbar_buttons/toggle_log_win_button.gif) | Log Window | Toggles the visibility of the Log Window. By default, the Log Window is located in the bottom left portion of the interface. |
| ![Toggle Find Window](../../../helps/Toolbar_buttons/toggle_find_win_button.gif) | Find Window | Toggles the visibility of the Find Window. By default, the Find Window is located in the bottom right portion of the interface. |
