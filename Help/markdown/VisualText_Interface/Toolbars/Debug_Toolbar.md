# Debug Toolbar

![](../../../helps/VisualText_Interface/Toolbars/debug_toolbar0614.gif)

The Debug Toolbar provides controls to display parse trees, intermediate files and output files.

| **Button** | **Name** | **Description** |
| --- | --- | --- |
| ![Toggle Page Mode](../../../helps/Toolbar_buttons/toggle_page_mode_button.gif) | **Toggle Page Mode** | Maintains coordinated sets of documents. For example, when the user runs the analyzer, its corresponding output and tree windows are updated automatically. Similarly when selecting passes in the Ana Tab, Page Mode activates the Next Text and Previous Text buttons. |
| ![Previous Text](../../../helps/Toolbar_buttons/previous_text_button.gif) | **Previous Text** | Displays previous input text file according to the ordering in the Text Tab. Works in Page Mode to automatically update the open documents associated with the input text. |
| ![Next Text](../../../helps/Toolbar_buttons/next_text_button.gif) | **Next Text** | Displays the next input text file according to the ordering in the Text Tab. Works in Page Mode to automatically update the open documents associated with the input text. |
| ![View Current Text](../../../helps/Toolbar_buttons/view_current_text_button.gif) | **View Current Text** | Displays the currently selected input data file in the Workspace. |
| ![View Web Page](../../../helps/Toolbar_buttons/view_web_page_button.gif) | **View Web Page** | Displays Web page for the HTML file selected in the Text Tab. |
| ![View Pass File](../../../helps/Toolbar_buttons/view_pass_file.gif) | **View Pass File** | Displays the currently selected pass file in the analyzer sequence in the Workspace. |
| ![View Parse Tree](../../../helps/Toolbar_buttons/view_entire_tree_button.gif) | **View Parse Tree** | Displays the full parse tree of a parsed document in the Workspace. This includes character position and white spaces in the document. |
| ![View Concept Tree](../../../helps/Toolbar_buttons/view_concept_tree_button.gif) | **View Concept Tree** | Displays the concept tree of a parsed document in the Workspace. |
| ![Toggle Node Variables](../../../helps/Toolbar_buttons/toggle_node_variables.gif) | Toggle Node Variables | When selected, enables the display of node variables in the parse tree. |
| ![View Dump Files](../../../helps/Toolbar_buttons/view_dump_files_button.gif) | **View Dump Files** | Displays the output files associated with a parsed document in the Workspace. When there are multiple files, the pulldown tab to the right of the button lists available files. |
| ![View Output File](../../../helps/Toolbar_buttons/view_output_file_button.gif) | **View Output File** | Displays the output.txt file associated with parsed input document in the Workspace. |
