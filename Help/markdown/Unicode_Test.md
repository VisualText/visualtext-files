# Unicode_Test

**NOTE:** Some users find a Microsoft library missing when they run VisualText Unicode.  support@textanalysis.com can help with any problems.

Unicode_Test is an analyzer supplied with the release, at Program Files/TextAI/VisualText/apps.  It provides sample input files that can be loaded and processed to produce a tokenized parse tree for various test languages.  VisualTextU.exe, the version of VisualText that supports Unicode, should be invoked in order to run this analyzer.
