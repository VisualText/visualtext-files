# fprintnvar

## Purpose

**OBSOLETE.**  To **fileName**, print the value of the variable **var** in the **ord**th element's node.

## Syntax

```
fprintnvar(fileName, var, ord)
```

```
filename - type: str

var - type:

ord - type:
```
