# debug

## Purpose

Enable C++ source code debugging for VisualText developers.

## Syntax

```
NULL = debug()
```

## Returns

## Remarks

Enables VisualText developers to set a breakpoint in a particular rule, from a C++ development environment, e.g. MS Visual C++.

## Example

```
@POST

debug();

@RULES

_xNIL <- _xNIL @@
```
