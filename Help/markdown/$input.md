# $input

## Purpose

Get the full input path for the file.

## Syntax

```
returnedString = variableType("$input")
```

```
returnedString - type: str

variableType - type: G
```

## Returns

## Remarks

## Example

$input returns "D:\apps\Resume\input\Dev1\rez.txt" if the input file is "rez.txt".
