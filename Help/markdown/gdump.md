# gdump

## Purpose

Dump all the global variables and their values to **fileName.**

## Syntax

```
gdump(fileName)
```

```
fileName - type: str
```

## Returns

## Remarks

The output file (**fileName**) must be set up in advance using **openfile**, **fileout** or an output statement.
