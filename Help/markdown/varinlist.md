# varinlist

## Purpose

Find var_str variable's value(s) in any pnode that matched elt_numth element.

## Syntax

```
returnedValue = varinlist(var_str, elt_num)
```

```
returnedValue - type: var value

var_str - type: str

elt_num - type: int
```
