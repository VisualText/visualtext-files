# opt

## Purpose

Match a minimum of 0 and a maximum of 1 node. Indicates element is optional.

## Remarks

Short for min=0 max=1.  **option**, **optional** and **o** are alternate forms for opt.

## Example

```
_vgroup <- _modal [opt]
 _have [opt] _be [opt]
 _verb @@
```
