# $inputname

## Purpose

Get the name of the input file.

## Syntax

```
returnedString = variableType("$inputname")
```

```
returnedString - type: str

variableType - type: G
```

## Returns

## Remarks

## Example

$inputname returns "rez.txt" if the input file is "rez.txt".
