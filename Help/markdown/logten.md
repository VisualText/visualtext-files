# logten

## Purpose

Compute base 10 logarithm of given number.

## Syntax

```
returnedFlt = logten(numFlt)
```

```
returnedFlt - type: float

numFlt - type: float
```

## Returns

Base 10 logarithm of number.

## Remarks

Analogous to log10 function in C/C++.

## Example

```
@CODE

"output.txt" << logten(3.1) << "\n";

@@CODE
```

```
Outputs something like:
```

```
.49136
```
