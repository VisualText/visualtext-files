# today

## Purpose

Format the current date and time as a string.

## Syntax

```
returnedStr = today()
```

```
returnedStr - type: str
```

## Returns

Return the current date and time as a string.

## Remarks

An "escape to the shell" from inside NLP++.

## Example

```
@CODE

"output.txt" << today() << "\n";

@@CODE
```
