# Table of Math Functions

NLP++ now features mathematical functions analogous to those provided in C/C++.

| **FUNCTION NAME** | **RETURNS** | **DESCRIPTION** |
| --- | --- | --- |
| [**abs(num_int)**](abs.md) | **INT** | Get the absolute value of a given integer. |
| [**factorial(num_int)**](factorial.md) | **INT** | Get the factorial of a non-negative integer. |
| [**logten(num_float)**](logten.md) | **FLOAT** | Get base 10 logarithm of given number. Analogous to log10 function in C/C++. |
| [**mod(num_int1, num_int2)**](mod.md) | **INT** | Get the remainder of dividing one integer by another. |
| [**randomint(num_int1, num_int2)**](randomint.md) | **INT** | Get a random integer between two given integers, inclusive. |
|   |   |   |
