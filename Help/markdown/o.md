# o

## Purpose

Match a minimum of 0 and a maximum of 1 node. Indicates element is optional.

## Remarks

Short for min=0 max=1.  **opt**, **option** and **optional** are alternate forms for o.

## Example

```
_vgroup <- _modal [o]
 _have [o] _be [o]
 _verb @@
```
