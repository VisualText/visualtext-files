# strnotequalnocase

## Purpose

Check if string 1 differs from string 2, ignoring letter case.

## Syntax

```
returnedBoolean = strnotequalnocase(wordString1, wordString2)
```

```
returnedBoolean - type: bool

wordString1 - type: str

wordString2 - type: str
```
