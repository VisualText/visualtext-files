# nodeconcept

## Purpose

Fetch the concept that the given node is a proxy for.

## Syntax

```
returnedConcept = nodeconcept(node)
```

```
returnedConcept - type: con

node - type: con
```

## Returns

The old functionality of this has been usurped by the new function *nodeowner*.
