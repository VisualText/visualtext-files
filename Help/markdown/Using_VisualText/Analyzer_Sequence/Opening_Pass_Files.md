# Opening Pass Files

**To open a pass file associated with a pass in the analyzer sequence**:

- In the **Ana Tab**, double-click on the **pass**.  The pass file associated with the pass opens in the Workspace.

| **Tip**: You can also right-click and select **Edit Pass File** from the pulldown menu. |
