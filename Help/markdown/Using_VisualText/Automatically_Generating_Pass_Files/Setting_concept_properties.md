# Setting Concept Properties

**To set concept properties for rule concepts**:

1. In the **Gram Tab**, select the **rule concept**.

1. Right click and select **Properties** from the pulldown menu. The Gram Concept Properties dialog is opened.

1. Check or uncheck the boxes for the appropriate attribute. (See [Setting Rule Generation Properties](../../Setting_Rule_Generation_Properties.md#Attributes) for a description of the attributes.)

1. Click **Done** to finish setting the properties.
