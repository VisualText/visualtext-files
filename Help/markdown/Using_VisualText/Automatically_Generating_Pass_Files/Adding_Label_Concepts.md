# Adding Label Concepts

Label concepts hold subsamples of larger concepts.  For example, the area code or exchange of a telephone number can be a subsample of a telephone number.

**To add a label concept to a rule concept**:

1. In the **Gram Tab**, select the rule concept where you want to add the label concept.

1. Right-click and select **Add** > **Label**.

1. Give the label concept a **name** in the dialog box.

1. Click **OK** to finish adding the label concept.

| **Note**: Label concepts can only be added to rule concepts. |
