# Deleting Text Files from a Project

**To delete a text file from your project**:

- In the **Text Tab**, right-click on the **file** you want to delete and select **Delete**. Any log files associated with the text file are also deleted.

| **Tip**: Delete log files from the Text Tab to save space in your analyzer project. |
