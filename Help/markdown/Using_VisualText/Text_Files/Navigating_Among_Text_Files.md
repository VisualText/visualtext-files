# Navigating Among Text Files

You can navigate among the different files in the Text Tab using the Page Mode feature.  This feature allows you to move from one file to the next file and is handy when you want to quickly get to the next file listed in the Text Tab.

To move from one file to the next in the Text Tab:

1. Double-click on a **file** to open it in the Workspace.

1. Select the **Page Mode Toggle**  ![](../../../helps/Toolbar_buttons/toggle_page_mode_button.gif)  to activate the yellow Previous Text ![](../../../helps/Toolbar_buttons/previous_text_button.gif) and Next Text ![](../../../helps/Toolbar_buttons/next_text_button.gif) arrows.

1. Use the Previous Text and Next Text arrows to open up files in the Workspace.

| **Note**: If a parse tree is displayed for the file, using the Previous Text and Next Text arrows will display the parse tree display for the next text in the list. |
