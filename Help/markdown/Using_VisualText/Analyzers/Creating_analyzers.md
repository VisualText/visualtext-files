# Creating Analyzers

**To create an analyzer**:

1. From the **File** menu, select **New Analyzer**.

1. In the **New Analyzer** dialog, give the analyzer a **name**. This name will be used to organize the analyzer project. VisualText automatically assigns a .ana extension to the name to mark it as the analyzer project file. You use the .ana file to open the analyzer project.

1. Specify the drive and folder where you want the analyzer project and its files to be created in **Location. **The application directory defaults to c:\apps. (You can change the default application directory in the Preferences dialog.)

1. For **Template**, select one of the templates under **Type**.  (For more on templates, see [Analyzer Templates](../../Analyzer_Templates.md#temp_types).)

1. Click **OK** to finish creating the analyzer project.
